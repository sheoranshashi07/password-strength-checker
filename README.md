# Password Strength Checker using Python

## Introduction

Passwords are an essential part of digital security. Many users create weak passwords that are easy to guess, which increases the risk of cyber attacks. This project was developed to help users evaluate the strength of their passwords and generate stronger alternatives.

## Project Objective

The main objective of this project is to create a simple Python-based tool that analyzes password strength based on common security rules such as password length, use of numbers, uppercase letters, and special characters.

## Features

- Checks password strength
- Detects weak passwords
- Generates strong random passwords
- Simple command-line interface
- Modular code structure

## Technologies Used

Python programming language

Libraries used:
- re (Regular Expressions)
- random
- string

## Project Structure

password-strength-checker/

password_generator.py – generates secure passwords  
utils.py – contains helper functions  
README.md – documentation of the project
